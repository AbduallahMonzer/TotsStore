# TotsStore

This repository contains a simple static website for a kids clothing catalog.

## How to use

1. Open `index.html` in your browser to view the catalog locally.
2. Edit `index.html` and `styles.css` to customize your products and style.

## Deploying with GitHub Pages

You can host this site for free using GitHub Pages:

1. Push this repository to GitHub.
2. Go to your repository **Settings** -> **Pages**.
3. Under **Source**, choose the branch (e.g. `main`) and select the root directory.
4. GitHub will provide a link like `https://<username>.github.io/TotsStore/` where your site will be live.

Orders can be handled via the email link in the page footer.
